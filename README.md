# Camera Shutter Count

A web-based tool to check your camera's shutter count directly in your browser using WebUSB.

## About

This project is inspired by [Free Shutter Counter](https://orlv.github.io/freeshuttercounter/) and adapted from the [libgphoto2](http://www.gphoto.org) library to use [WebUSB](https://developer.mozilla.org/en-US/docs/Web/API/WebUSB_API).

## Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. Every push to the `main` branch triggers a build and deployment.
